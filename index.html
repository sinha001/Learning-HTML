Non-semantic Tags

  1. Div Tag:

    Div is a container used for other HTML elements. It is a Block elements(takes full width)

  2. Span Tag:

    Span is also a container used for other HTML elements. It is a Inline Element(takes width as per size)

Tables in HTML

Tabels are used to represent real life table data
  • tr tag: used to display table row
  • td tag: used to display table data
  • th tag: used to display table header

thead & tbody in HTML

  • thead tag: used to wrap table head
  • tbody tag: used to wrap table body

Form in HTML

Forms are used to collect data from the user. Eg: sign up / login / help requests / contact me

  1. Action in Form

    Action attribute is used to define what action needs to be performed when a form is submitted. eg: form action="/action.php"
